# Automyx 2.5 - Modelo de Ollama

![Automyx Logo](assets/logo.png)

## ¿Qué es Automyx?

Automyx es un conjunto de herramientas de automatización profesional con inteligencia artificial con control completo del sistema, edición 2D/3D y voz en tiempo real.

Este modelo de Ollama está diseñado para ser el cerebro de Automyx, con una personalidad profesional y amigable, listo para ayudarte en tus proyectos de automatización.

## Características del modelo

- **Nombre: Automyx 2.5
- **Basado en: llama3.2:1b
- **Temperatura: 0.7 (equilibrado entre creatividad y coherencia)
- **Licencia: MIT

## Cómo usar este modelo

### 1. Instalar y ejecutar el modelo
```bash
ollama pull NEXORATECHNOLOGYGEO/automyx
```

### 2. Ejecutar el modelo
```bash
ollama run NEXORATECHNOLOGYGEO/automyx
```

### 3. Usar el modelo con Automyx
```bash
python automix.py ollama launch --model NEXORATECHNOLOGYGEO/automyx --location local
```

## ¿Qué puede hacer Automyx?

- **Automatización del sistema (comandos de terminal, gestión de archivos, apertura de aplicaciones)
- **Edición profesional de imágenes
- **Modelado y renderizado 3D con Blender
- **Gestión de proyectos (detección de errores, autocorrección, documentación)
- **Interacción de voz en tiempo real
- **Y mucho más!
