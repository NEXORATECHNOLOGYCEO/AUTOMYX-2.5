# Cómo Contribuir a Automyx

¡Gracias por tu interés en contribuir a Automyx! Toda ayuda es bienvenida.

## Pasos para Contribuir

1. **Fork el repositorio**
   Haz click en el botón "Fork" en la parte superior derecha de la página del repositorio.

2. **Clona tu fork**
   ```bash
   git clone https://github.com/TU-USUARIO/AUTOMYX-2.5.git
   cd AUTOMYX-2.5
   ```

3. **Crea una rama para tu característica o corrección**
   ```bash
   git checkout -b feature/nueva-caracteristica
   ```
   O si es una corrección de bugs:
   ```bash
   git checkout -b fix/correccion-bug
   ```

4. **Haz tus cambios y commit**
   ```bash
   git add .
   git commit -m "Descripción clara de tus cambios"
   ```

5. **Push tu rama**
   ```bash
   git push origin feature/nueva-caracteristica
   ```

6. **Abre un Pull Request**
   Ve a la página de tu fork en GitHub y haz click en "Compare & pull request".

## ¿Qué Puedes Contribuir?

- Reportar bugs
- Sugerir nuevas características
- Mejorar la documentación
- Corregir errores de escritura
- Escribir tests
- Y más!

## Código de Conducta

Por favor, sé respetuoso con todos los colaboradores.

¡Gracias por hacer de Automyx un mejor proyecto! 🚀
