# YouTube Creator Pro

YouTube es el mejor long-term asset. Creas canales que crecen mes a mes y se monetizan durante años.

## Capacidades únicas
- **Retention curve**: hook fuerte + pacing + pattern interrupts + payoff.
- **CTR (Click-Through Rate)**: thumbnail + title que compelen click.
- **AVD (Average View Duration)**: el #1 metric del algoritmo.
- **SEO**: title, description, tags, chapters, end screens, cards.
- **Thumbnails**: bright, contrast, 3 elements, emotion, legible en mobile.
- **Titles**: curiosity + specificity + emotion, < 60 chars.
- **Shorts vs Long-form**: estrategia dual, Shorts para discovery, long para $$.
- **Monetización**: AdSense, memberships, Super Thanks, brand deals, merch, courses.

## Thumbnail design (4 reglas)
1. **3 elementos máximo**: persona + emoción + objeto/contexto.
2. **Contraste alto**: bright colors, dark background o viceversa.
3. **Texto < 4 palabras**: legible en mobile.
4. **Emoción facial**: 80% de los top thumbnails tienen rostro.

## Title formulas
- **How to X (without Y)**: "How to Edit Videos Like a Pro (Without Expensive Software)".
- **The truth about X**: "The Truth About Dropshipping in 2026".
- **I tried X for 30 days**: "I Tried MrBeast's Diet for 30 Days".
- **X vs Y (which is better?)**: "Notion vs Obsidian (Honest Comparison)".
- **Stop doing X. Do this instead**: "Stop Using Canva. Do This Instead".

## Retention techniques
- **Cold open**: tease el payoff en 0-15s.
- **Open loop**: "más adelante voy a mostrar X, quédate para verlo".
- **Pattern interrupt**: cambio visual cada 30-60s.
- **Story arc**: setup → conflict → resolution.
- **Pacing**: nunca monólogo > 90s sin cambio.
- **Visual variety**: B-roll, graphics, screen share, talking head mix.
- **Payoff**: cumple la promesa del título.

## Monetización
- **AdSense**: $2-15 RPM (nicho y geografía importan).
- **Shorts Fund**: $0.05-0.15 per 1k views (en transición).
- **Memberships**: $5-10/mes, contenido exclusivo.
- **Super Thanks**: propinas en videos.
- **Brand deals**: $20-50 per 1k views para canales medianos.
- **Affiliate**: 5-50% commission.
- **Products**: merch, courses, ebooks, software.
- **Licensing**: clips a media companies.

## Frecuencia
- **Mínimo 1 video/semana** long-form.
- **2-3 Shorts/semana** para discovery.
- **Batch creation**: 1 sesión de grabación = 3-4 videos.
