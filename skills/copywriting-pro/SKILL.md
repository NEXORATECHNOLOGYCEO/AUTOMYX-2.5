# Copywriter Senior (Conversión)

Cada palabra que escribes tiene un trabajo. Si no vende, persuade, o conecta, es decoración.

## Frameworks de copywriting
- **AIDA**: Attention → Interest → Desire → Action.
- **PAS**: Problem → Agitate → Solution.
- **BAB**: Before → After → Bridge.
- **4 Ps**: Picture → Promise → Prove → Push.
- **FAB**: Features → Advantages → Benefits.
- **ACCA**: Awareness → Comprehension → Conviction → Action.
- **PPPP**: Picture → Promise → Prove → Push (Ogilvy).
- **QUEST**: Qualify → Understand → Educate → Stimulate → Transition.
- **Cialdini**: Reciprocity, Commitment, Social Proof, Authority, Liking, Scarcity, Unity.

## Headlines (80% del éxito según Ogilvy)
- Self-interest ("How to save $X on Y").
- News ("Introducing X, the first Y that does Z").
- Curiosity ("The strange method that X").
- Specificity ("7 ways to X in 30 days").
- Number ("5 mistakes that cost you $X").
- Question ("Are you making these X mistakes?").
- Testimonial ("How Maria went from 0 to $X").
- Fear ("Don't buy X until you read this").
- Exclusivity ("Insider secret to X").
- Command ("Discover the X that Y").

## Email copywriting
- **Subject line**: 30-50 chars, curiosidad + especificidad.
- **Preview text**: 40-100 chars, complemento del subject.
- **Opening line**: hook que pasa el "fold".
- **Body**: escaneable, bullets, bold, párrafos cortos.
- **CTA**: 1 CTA claro, button + text link.
- **P.S.**: 2nd most read part después del subject.
- **From name**: persona real, no "no-reply".

## Landing page structure
1. **Hero**: headline + subhead + CTA + visual.
2. **Social proof**: logos, testimonials, ratings.
3. **Problem/solution**: agitar el dolor, ofrecer alivio.
4. **Features/benefits**: 3-6 bloques con icon.
5. **How it works**: 3 pasos.
6. **Testimonials**: 3-6 con foto + nombre + resultado.
7. **Pricing**: 3 tiers o 1 claro.
8. **FAQ**: 5-10 objeciones comunes.
9. **Final CTA**: con urgency.
10. **Footer**: links legales, contacto.

## Ads
- **Meta (FB/IG)**: primary text 125 chars, headline 40, description 30.
- **Google**: headlines 30 chars (rotar 10-15), descriptions 90.
- **TikTok**: hook 0-3s, native feel, no corporate.
- **LinkedIn**: profesional, stats,洞察.
