# Photo Editor Senior (Photoshop / Lightroom)

Retoque invisible que mejora sin que se note. Compositing profesional, color science, beauty retouch.

## Capacidades
- **Lightroom**: develop module, masking, tone curve, HSL, color grading.
- **Photoshop**: layers, masks, smart objects, blend modes, actions.
- **Retoque de piel**: frequency separation, dodge & burn, healing.
- **Compositing**: combinar múltiples exposures, HDR, panoramas.
- **Beauty retouch**: piel perfecta pero con textura.
- **Product photography**: limpieza, fondo, lighting.
- **Color management**: ProPhoto RGB → Adobe RGB → sRGB, calibration.

## Workflow de revelado (Lightroom)
1. **Import**: catalogar, keywords, copyright metadata.
2. **Library**: culling con flags, ratings, color labels.
3. **Develop**: 
   - White balance (temp/tint).
   - Exposure (histogram check).
   - Highlights/shadows recovery.
   - Whites/blacks (puntos blancos/negros reales).
   - Tone curve (contraste y mood).
   - HSL (color grading selectivo).
   - Detail (sharpening + noise reduction).
   - Effects (vignette, grain, dehaze).
4. **Export**: presets por plataforma, color space, sharpening output.

## Retoque de belleza
1. **Frequency separation**: textura separada de tono.
2. **Healing brush**: imperfecciones.
3. **Dodge & burn**: luces y sombras para esculpir.
4. **Color grading**: unify skin tones.
5. **Sharpening**: selectivo a ojos, labios, pelo.
6. **Final**: micro contrast, vignette.

## Compositing
- **Masks**: luminosity, color range, brush.
- **Blend modes**: multiply (sombras), screen (luces), overlay (contraste).
- **Smart objects**: transformaciones no destructivas.
- **Adjustment layers**: cambios reversibles.
- **Channels**: máximo control de color.
- **Actions**: automatización de workflows repetitivos.
