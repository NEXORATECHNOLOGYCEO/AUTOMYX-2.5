El Gateway: El Corazón de Automyx

El Gateway es el componente central que une todo. Es el servidor que se ejecuta en tu máquina y actúa como intermediario entre:

  • La Interfaz Web (Dashboard): Donde tú interactúas con el chat, configuras agentes y usas las habilidades.
  • Los Agentes y LLMs: Los modelos de IA que generan las respuestas y toman decisiones.
  • Las Habilidades y Herramientas: Todo lo que permite a Automyx interactuar con el mundo real (ejecutar comandos, editar archivos, navegar por la web, etc.).
  • Tu Sistema Operativo: Windows, macOS o Linux, donde se ejecutan todas las acciones.

¿Para qué es el Token?

El token es tu llave de seguridad personal. Garantiza que solo tú (o quien tenga el token) pueda acceder y controlar tu Gateway. Es generado aleatoriamente cada vez que inicias el servidor y se muestra en la terminal.
